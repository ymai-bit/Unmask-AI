# Unmask-AI

## What This Does
This tool detects potentially misleading or scam skincare products by analyzing product descriptions, claims, and reviews.

## Why It Matters
AI is increasingly used to generate fake reviews and deceptive marketing. This project helps users identify those risks.

## How It Works
- User inputs product info
- LLM evaluates claims, ingredients, and trust signals
- Outputs a scam risk score and explanation

## How to Run
npm install
npm run dev
